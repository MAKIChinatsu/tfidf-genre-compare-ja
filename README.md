# tfidf-genre-compare-ja
TF-IDF comparison script for pre-tokenized Japanese corpora (poetry/tales)
# TF-IDF Genre Comparison for Japanese Tokenized Corpora

日本語の分かち書き済みコーパスを対象に、ジャンル内（詩／童話）で作家ごとのTF-IDF特徴語を比較するためのスクリプトです。

## 概要

このリポジトリには、以下の処理を行う Python スクリプトを含みます。

- 分かち書き済みテキストの読み込み
- ストップワード除去
- 記号・数字トークンの除去（設定可）
- 一字語の制御（設定により「漢字一字のみ許可」）
- ジャンルごとの TF-IDF 計算（詩と童話を分けて計算）
- 作家ごとに文書単位TF-IDFを平均（mean）し、上位語を出力
- 診断用の文書分割情報（分割数・トークン数など）を出力

## 想定する入力データ

- **分かち書き済み**の日本語テキスト（空白区切り）
- 形態素解析・原形化済みテキストを想定
- ストップワードリスト（1行1語）

## 分割方法（このスクリプトの前提）

- **詩コーパス**：`★` を作品区切りとして分割
- **童話コーパス**：固定トークン数（例：500語）で人工的に分割

※ 童話は作品単位の区切りがない場合を想定し、トークン数分割に対応しています。

## 注意（データ公開について）
